Share from iOS — one-time setup
iOS doesn't support Web Share Target, so install this Shortcut instead — it adds Overseer to every app's share sheet.
  1. 1Open the Shortcuts app → tap + to create a new shortcut.
  2. 2Tap Add Action → search "URL" → select the URL action. Paste this into the URL field:
  3. 3Tap + → search "Open URLs" → add it. It will automatically use the URL from step 2.
  4. 4Tap the shortcut name at the top → rename it Save to Overseer.
  5. 5Tap ··· (top right) → enable "Show in Share Sheet" → done.
Now when you're on a TikTok, Instagram post, or any link — tap Share → Save to Overseer. It opens Overseer with the link pre-filled and Claude auto-categorizes it.
